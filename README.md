# Book Store API

## Requirements

- **Node.js** version **v22.14.0** (or higher recommended)
- **npm** package manager (comes with Node.js)

---

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/avasicquipu/book_store_api.git
   cd book_store_api
   ```

2. Install dependencies:

   ```bash
   npm i
   ```

3. Run the server:
   ```bash
   npm start
   ```
